# AI Burnout Monitor

## Overview
A web application that analyzes user input to detect burnout or stress levels using AI APIs.

## Features
- Takes user input related to mood and stress
- Uses API integration to analyze burnout levels
- Displays results in a simple and interactive UI

## Tech Stack
- React.js
- JavaScript
- Node.js(for API handling)
- Groq API(AI-based analysis)

## What I Learned
- Working with API keys and external services
- Handling user input and responses in React
- Building real-world use cases using AI concepts
